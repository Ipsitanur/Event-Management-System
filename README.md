# Event Management System

A console-based Event Management System developed in C, featuring role-based access for Admin and User, allowing efficient management and booking of event services.

## Features

### Admin
- Add new event/service details
- View all events/services
- Update existing event/service information
- Delete events/services

### User
- Browse and select event categories
- Select services based on preference
- Book selected services/events
- View generated bill after booking
- Submit reviews/feedback for services

## Technologies Used
- Language: C
- Concepts: Structures, File Handling, Functions, Role-based Logic, Loops, Conditional Statements

## How to Run
1. Clone this repository:
   `https://github.com/Ipsitanur/Event-Management-System.git`
2. Navigate to the project folder:
   `cd Event-Management-System`
3. Compile the code:
   `gcc main.c -o event_management`
4. Run the executable:
   `./event_management`
